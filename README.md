Zoo Map Website  

This is a simple responsive website that provides an interactive zoo map along with useful sections such as Animals, Places, Weather, and Feedback.  

## 📂 Project Structure  

project-folder/
│── index.html        # Main homepage with zoo map
│── animal.html       # Page showcasing animals
│── places.html       # Page for different places in the zoo
│── weather.html      # Page showing weather info
│── feedback.html     # Feedback form page
│── style.css         # Stylesheet for the website
│── logo.png          # Website logo
│── map.png           # Zoo map image

## 🚀 Features  
- **Header with Logo** – Displays the site’s branding.  
- **Top Navigation Tabs** – Quick links to Animals, Map, and Places pages.  
- **Main Map Section** – Displays an image of the zoo map.  
- **Bottom Navigation Tabs** – Links to Weather updates and Feedback form.  
- **Responsive Design** – Works across desktop and mobile devices.  

## 📖 How to Use  
1. Open `index.html` in your browser to view the zoo map.  
2. Navigate using the top and bottom tabs to explore Animals, Places, Weather, and Feedback.  
3. Customize `style.css` to change the look and feel of the site.  
4. Replace `logo.png` and `map.png` with your own images if needed.  

## 🔧 Setup  
No installation required. Simply download the project files and open `index.html` in any web browser.  

## 📌 Notes  
- Ensure that all linked files (`style.css`, images, and other HTML pages) are in the correct directory.  
- The project can be hosted on any web server or opened locally.
